Overflow The factorial of a number grows quite fast. For example, the factorial of 20 is 2,432,902,008,176,640,000. That’s already bigger than the maximum value for a 32-bit integer, 2,147,483,647. When the result of an arithmetic operation is bigger than the maximum value for a given integer type, we are talking about an integer overflow. Integer overflows are an issue because they violate the contract for arithmetic operations.
